# DI-Web-Analytics

### Build project for npm package

```bash
yarn build
```

#### 🚀 Deploy

```bash
npm publish
```

### Build project with webpack

⚠️ M1 CPU cant run, please run with intel CPU

#### 🛵 Development

```bash
./build_webpack.sh development
```

#### 🏍️ Production

```bash
./build_webpack.sh production
```

### Run test case

```bash
yarn test
```
